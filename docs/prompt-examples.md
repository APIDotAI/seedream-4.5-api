# Prompt examples for Seedream 4.5

These prompts are written for testing the Seedream 4.5 API through APIDot. They are starting points for product teams, not guaranteed outputs. Adjust brand names, subject details, source URLs, and safety rules before production use.

## 1. 4K campaign visual

A premium product campaign image of a matte ceramic mug on a warm gray studio background, precise typography on the mug reading APIDot, soft window light, balanced shadows.

## 2. Custom-size editorial

A vertical fashion editorial portrait in a clean studio with refined lighting, strong composition, readable magazine-style cover typography, production-ready 4K detail.

## 3. Reference edit

Keep the product shape and label from the source image, replace the background with a clean winter studio setup, preserve realistic shadows, ecommerce hero asset.

## 4. Multi-image consistency

Create a consistent set of three product images for the same ceramic candle jar: front hero, lifestyle table scene, close-up label detail, matching lighting and color.

## 5. Ad poster typography

A polished ad poster for a premium tea brand with readable headline 'Quiet Morning', elegant package design, natural paper texture, warm studio light.

## 6. Material replacement

Edit the source product so the plastic shell becomes brushed aluminum while preserving exact silhouette, label placement, camera angle, and shadow direction.

## Production prompt notes

- Store prompt versions together with task_id, model, user/job owner, and request payload.
- Keep source media URLs reachable long enough for APIDot to fetch them.
- Be explicit about composition, typography, material, lighting, output size, and reference-image constraints.
- Do not log prompts or source media URLs if they may contain private customer data.
